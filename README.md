# Sphere AI Assistant

Локальное настольное приложение с AI-ассистентом на основе **Flet** (Flutter на Python).  
Все данные хранятся локально. Приоритет на приватность, скорость и кроссплатформенность.

## Возможности

- **Чат с ИИ** — общение с локальной моделью через Ollama или ~~OpenAI API~~
- **Заметки** — Markdown-редактор с live-предпросмотром, папками и тегами
- **Задачи** — канбан-доска (To Do / In Progress / Done) с приоритетами и дедлайнами
- **Календарь** — управление событиями с навигацией по месяцам
- **База знаний** — загрузка документов (PDF, DOCX, MD) и Q&A по ним (RAG)
- **Нейронный поиск** — семантический поиск по всем данным через ChromaDB
- **Уведомления** — локальные (macOS) и удалённые (Telegram)
- **Тёмная/светлая тема** — с поддержкой Material Design 3

## Быстрый старт

### 1. Установка зависимостей

```bash
# Создайте виртуальное окружение
python -m venv venv
source venv/bin/activate  # macOS/Linux
# или: venv\Scripts\activate  # Windows

# Установите зависимости
pip install -r requirements.txt
```

### 2. Установка Ollama (для локальных моделей)

```bash
# macOS — скачайте с https://ollama.ai/
# Или через Homebrew:
brew install ollama

# Загрузите модель:
ollama pull llama2
# или
ollama pull mistral

# Запустите сервер:
ollama serve
```

### 3. Запуск приложения

```bash
python main.py
```

Или в режиме разработки с горячей перезагрузкой:

```bash
flet run main.py
```

## Настройки

Конфигурация хранится в `data/config.yaml` и настраивается через интерфейс (раздел «Настройки»):

- **AI-провайдер** — Ollama (локально) или ~~OpenAI (облако)~~
- **Модель** — выбор модели (llama2, mistral, ~~gpt-3.5-turbo~~ и др.)
- **Температура** — креативность ответов (0.0 — точные, 1.0 — творческие)
- **Telegram** — токен бота и chat_id для удалённых уведомлений
- **Тема** — тёмная или светлая
